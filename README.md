# Clio

### Build

`sbt compile`

### Run tests

`sbt test`

### Quickly run a server

`sbt run`

### Connect to the running server

`curl localhost:8080`

### Build an executable jar

`sbt assembly`

### Build docker image

`sbt docker`

### Run the docker image

`docker run --rm -it -p 8080:8080 broadinstitute/clio:<version>`
